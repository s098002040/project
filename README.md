# 專案名稱blogger
此為簡易的部落格專案

## 專案介紹
此專案為簡易的Bloger，包含前台流程系統，基於Java、Spring4與Hibernate框架實現。
- 部落格系統
  - 登入
  - 註冊
  - 開帖
  - 回帖
  - 瀏覽
- 後台管理系統
  - 點擊次數
  - 回復次數
  - 初始化資料
 
## 使用到的技術
| 技術                | 說明       |
|-------------------|----------| 
| Java        		| Web應用開發  | 
| Spring4        	| framwork  | 
| Hibernate	        | ORM框架    | 
| SQL Server		| 資料庫      | 
| JS、jQery			| 前端      |
